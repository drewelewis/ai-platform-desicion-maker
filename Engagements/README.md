# Engagements Folder Guide

Use this folder for customer-specific decision packages.

## Folder Naming
Use one folder per decision cycle:
- <customer-slug>-<yyyymmdd>

Example:
- contoso-20260709

## Minimum Required Files Per Engagement
- intake-worksheet.md
- recommendation-memo.md

## Optional Files
- notes.md
- artifacts/ (screenshots, exports, references)

## Rules
- Do not edit master templates in Templates for customer-specific content.
- Every risk, dependency, and assumption must have owner and due date.
- Keep all engagement artifacts in this folder for auditability.
